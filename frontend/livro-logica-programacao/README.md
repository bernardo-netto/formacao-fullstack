# 🚀 Pense em Código — Landing Page Acadêmica

Landing page moderna, responsiva e semântica desenvolvida com HTML5, CSS3 e JavaScript puro.

O projeto foi criado com foco em performance, acessibilidade, experiência visual premium e boas práticas modernas de desenvolvimento Front-End.

---

## 📚 Sobre o Projeto

A aplicação apresenta uma experiência visual acadêmica voltada para lógica de programação, desenvolvimento web e formação profissional.

O layout foi desenvolvido utilizando arquitetura semântica, animações suaves, responsividade avançada e separação organizada de estilos e scripts.
