# 🎓 EduTrack — Plataforma Educacional Inteligente

Projeto de landing page moderna desenvolvido com HTML5, CSS3 e JavaScript puro.

A proposta da EduTrack é apresentar uma plataforma focada em gestão educacional técnica e profissionalizante, utilizando uma interface moderna, responsiva e com experiência visual premium.

---

# 🚀 Tecnologias Utilizadas

- HTML5 Semântico
- CSS3 Moderno
- JavaScript Vanilla
- Responsividade Mobile First
- CSS Grid & Flexbox
- Animações CSS
- Variáveis CSS (Custom Properties)

---

# 🎨 Funcionalidades

✅ Layout moderno estilo SaaS  
✅ Responsividade completa  
✅ Menu mobile hamburger  
✅ Modal de autenticação  
✅ Botão voltar ao topo
