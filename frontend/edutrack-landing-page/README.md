🚀 EduTrack - Landing Page

Gestão de aprendizado inteligente para escolas técnicas.

Este projeto é uma Landing Page moderna e responsiva desenvolvida para a plataforma EduTrack, focada em centralizar avaliações, presença e progresso de alunos em uma interface intuitiva.

🎯 Objetivo do Projeto
O EduTrack visa transformar a maneira como escolas técnicas operam, eliminando burocracias e fornecendo dados em tempo real para decisões pedagógicas precisas.

🛠️ Tecnologias Utilizadas

HTML5: Estruturação semântica do conteúdo.
CSS3: Estilização moderna (Flexbox/Grid) e responsividade.
JavaScript: Interações e dinamismo da página.
Google Fonts: Tipografia personalizada para melhor leitura.

🌟 Os 4 Pilares do EduTrack

Conforme apresentado na plataforma, o sistema se baseia em:

1. Foco Técnico: Módulos específicos para laboratórios e oficinas.

2. Dados Acadêmicos: Dashboards que mostram o desempenho individual.

3. Integração Total: Comunicação fluida entre coordenação e alunos.

4. Segurança Robusta: Dados protegidos com criptografia de ponta.
